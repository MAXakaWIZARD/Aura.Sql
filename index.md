Getting Started
---------------

The Aura.Sql package will provide adapters for connecting to these relational database systems:

- Microsoft SQL Server (Windows only)
- MySQL
- Oracle
- Postgres
- Sqlite 2
- Sqlite 3

The adapters are being ported and refactored from the Solar_Sql adapters.
